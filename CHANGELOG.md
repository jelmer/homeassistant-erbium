# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-10

### Added
- Initial release of Erbium Home Assistant addon
- DNS server support
- DHCP server support
- Router Advertisements support (configurable)
- Web UI for monitoring DHCP leases and DNS records
- Flexible configuration through Home Assistant UI
- Support for custom TOML configuration
- Multi-architecture support (aarch64, amd64, armhf, armv7, i386)

### Notes
- This is an early release based on Erbium beta
- Web UI currently shows placeholder data (TODO: integrate with Erbium state)
